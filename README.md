## Features:

###### See and compare weather in different cities
###### Shows Temperature	Pressure	Humidity

## To run app:
Install npm and node
https://www.npmjs.com/get-npm

```
   npm install  
```
```
   npm start
```
### See website on http://localhost:3000/

### Technologies:
React.js, Es6, Axios, Openweather API.
